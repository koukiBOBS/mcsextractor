# Minecraft 基岩版结构提取工具

这是一个功能完整的网页应用，用于从 **Minecraft 基岩版**的 `.mcworld` 世界存档文件中提取结构数据。

## 🎯 主要功能

- ✅ **上传支持**: 支持拖拽或点击选择 `.mcworld` 文件上传
- ✅ **自动解析**: 自动解析存档内部的 LevelDB 数据库
- ✅ **结构筛选**: 筛选并提取所有 `mystructure:` 命名空间保存的结构
- ✅ **格式验证**: 对每条数据进行 NBT 格式校验，确保有效性
- ✅ **批量导出**: 将所有提取的结构打包成 ZIP 压缩包自动下载
- ✅ **详细日志**: 实时显示解析进度、发现的结构名称、文件大小和错误信息

## 🚀 快速开始

### 方式 1: 直接打开 HTML 文件

1. 在浏览器中打开 `index.html` 文件
2. 点击上传区域或拖拽 `.mcworld` 文件
3. 点击"开始提取"按钮
4. 等待处理完成，查看提取的结构
5. 点击"⬇️ 下载 ZIP"下载结构文件

### 方式 2: 使用本地服务器（推荐）

使用 Python：
```bash
python -m http.server 8000
```

或使用 Node.js：
```bash
npx http-server
```

然后在浏览器访问：
```
http://localhost:8000
```

## 📊 工作原理

### 1. 文件上传
- 通过 FileAPI 读取用户选择的 `.mcworld` 文件
- 验证文件类型和大小（限制 500MB）

### 2. ZIP 解压
- 使用 JSZip 解压 `.mcworld` 文件
- 列出所有内部文件和文件夹

### 3. 结构提取
提取采用多层级方法：
- **方法 1**: 直接查找 `.mcstructure` 文件
- **方法 2**: 从 LevelDB 数据库（`db/` 文件夹）中查找结构数据
- **方法 3**: 解析 `level.dat` NBT 文件
- **方法 4**: 深度扫描所有文件查找 NBT 格式的结构

### 4. 格式验证
- 检查是否为有效的 NBT 格式
- 验证 NBT 标签类型字节

### 5. 打包下载
- 创建新的 ZIP 文件
- 包含所有提取的结构
- 生成详细的提取报告

## 📋 NBT 格式说明

NBT (Named Binary Tag) 是 Minecraft 使用的数据格式。

| 类型值 | 类型名 | 说明 |
|-------|--------|------|
| 0x01 | TAG_Byte | 单字节整数 |
| 0x02 | TAG_Short | 短整数（2字节） |
| 0x03 | TAG_Int | 整数（4字节） |
| 0x08 | TAG_String | 字符串 |
| 0x0a | TAG_Compound | 复合标签 |

## 🐛 故障排除

### 找不到任何结构

**检查清单**:
- 确保文件是有效的基岩版 `.mcworld` 文件
- 结构必须使用结构方块并以 `mystructure:` 命名空间保存
- 查看操作日志了解具体原因

### 下载的 ZIP 文件是空的

可能是提取失败。检查日志中的错误信息。
